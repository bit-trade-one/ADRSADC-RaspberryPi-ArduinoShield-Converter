# ADADRSADC Raspberry Pi用Arduinoシールド変換基板

![](https://bit-trade-one.co.jp/wp/wp-content/uploads/2018/05/2048bd17b54e1d33844f3687563ff731.png)

ラズベリーパイ上で豊富に用意された Arduinoシールドを使うための変換基板です。  
Arduino UNO の信号をレベル変換しラズベリーパイに接続、  
豊富なArduino UNO用シールドを活用できます。（Arduino UNO Rev3対応）  
また Grove システム対応のコネクタ 2 つを搭載、市販の Grove システムに対応したサーボやセンサ、  
アクチュエータを利用できるなど拡張性を活かした設計のため、プロトタイピングにも最適です。
